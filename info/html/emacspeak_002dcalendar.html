11.35 emacspeak-calendar

This module speech enables the Emacs Calendar. Speech enabling is not the same as speaking the screen: This is an excellent example of the advantages of speech-enabled interaction.

11.35.1 emacspeak-calendar Commands

Interactive Command: emacspeak-appt-repeat-announcement nil

C-e A

Speaks the most recently displayed appointment message if any.

Interactive Command: emacspeak-calendar-setup-sunrise-sunset nil

Set up geo-coordinates using Google Maps reverse geocoding. To use, configure variable gweb-my-address via M-x customize-variable.

Interactive Command: emacspeak-calendar-speak-date nil

Speak the date under point when called in Calendar Mode.

11.35.2 emacspeak-calendar Options

Variable: User Option emacspeak-calendar-mark-personality

Personality to use when showing marked calendar entries.
