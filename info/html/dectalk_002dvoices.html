11.5 dectalk-voices

This module defines the various voices used in voice-lock mode. This module is Dectalk specific.

11.5.1 dectalk-voices Commands

Interactive Command: dectalk

C-e d C-d

Select Dectalk TTS server.

Interactive Command: dtk-exp

Select Dectalk TTS server.
