11.11 emacspeak-2048

Commentary: Speech-enable 2048 Game

11.11.1 emacspeak-2048 Commands

Interactive Command: emacspeak-2048-pop-state nil

Reset state from stack.

Interactive Command: emacspeak-2048-push-state nil

Push current game state on stack.

Interactive Command: emacspeak-2048-randomize-game (&optional count)

Puts game in a randomized new state.

Interactive Command: emacspeak-2048-score nil

Show total on board.

Interactive Command: emacspeak-2048-speak-board nil

Speak board.

Interactive Command: emacspeak-2048-speak-transposed-board nil

Speak board column-wise.
