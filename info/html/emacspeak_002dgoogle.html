11.36 emacspeak-google

Commentary: There are a number of search tools that can be implemented on the Google search page — in a JS-powered browser, these show up as the Google tool-belt. This module implements a minor mode for use in Google result pages that enables these tools via single keyboard commands. Originally all options were available as tbs=p:v Now, some specialized searches, e.g. blog search are tbm=
