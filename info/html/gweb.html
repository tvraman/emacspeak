11.196 gweb

Provide Google services — such as search, search-based completion etc. For use from within Emacs tools. This is meant to be fast and efficient — and uses WebAPIs as opposed to HTML scraping.

11.196.1 gweb Commands

Interactive Command: gweb-google-at-point (search-term &optional refresh)

Google for term at point, and display top result succinctly. Attach URL at point so we can follow it later — subsequent invocations of this command simply follow that URL. Optional interactive prefix arg refresh forces this cached URL to be refreshed.

Interactive Command: gweb-news-view (query)

Display News Search results in a browser.
