11.30 emacspeak-feeds

Commentary: This module provides Feeds support for Emacspeak

Automatically generated documentation for commands defined in module emacspeak-feeds.

Interactive Command: emacspeak-feeds-add-feed (title url type)

C-e M-u

Add specified feed to our feed store.

Interactive Command: emacspeak-feeds-archive-feeds nil

Archive list of subscribed fees to personal resource directory. Archiving is useful when synchronizing feeds across multiple machines.

Interactive Command: emacspeak-feeds-atom-display (feed-url)

Display ATOM feed.

Interactive Command: emacspeak-feeds-browse (feed)

C-e C-u

Browse specified feed.

Interactive Command: emacspeak-feeds-fastload-feeds nil

Fast load list of feeds from archive. This directly updates emacspeak-feeds from the archive, rather than adding those entries to the current set of subscribed feeds.

Interactive Command: emacspeak-feeds-find-feeds (query)

Browse feeds matching specified query.

Interactive Command: emacspeak-feeds-lookup-and-view (site)

Lookup feed URL for a site and browse result.

Interactive Command: emacspeak-feeds-opml-display (feed-url)

Display OPML feed.

Interactive Command: emacspeak-feeds-restoere-feeds nil

Restore list of subscribed fees from personal resource directory. Archiving is useful when synchronizing feeds across multiple machines.

Interactive Command: emacspeak-feeds-rss-display (feed-url)

Display RSS feed.
