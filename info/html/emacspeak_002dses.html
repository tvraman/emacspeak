11.143 emacspeak-ses

ses implements a simple spread sheet and is part of Emacs 21.4 This module speech-enables ses

11.143.1 emacspeak-ses Commands

Interactive Command: emacspeak-ses-backward-column-and-summarize nil

Move to previous column and summarize.

Interactive Command: emacspeak-ses-backward-row-and-summarize nil

Move to previous row and summarize.

Interactive Command: emacspeak-ses-forward-column-and-summarize nil

Move to next column and summarize.

Interactive Command: emacspeak-ses-forward-row-and-summarize nil

Move to next row and summarize.

Interactive Command: emacspeak-ses-summarize-cell (cell-name)

Summarize specified cell.

Interactive Command: emacspeak-ses-summarize-current-cell (&rest ignore)

Summarize current cell.
