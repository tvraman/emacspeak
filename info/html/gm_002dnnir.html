11.196.1 gm-nnir Commands

Interactive Command: gm-nnir-group-make-gmail-group nil

Use GMail search syntax exclusively. See https://support.google.com/mail/answer/7190?hl=en for syntax. Default is to search All Mail when not on a Group line.

Interactive Command: gm-nnir-group-make-nnir-group nil

GMail equivalent of gnus-group-make-nnir-group. Default is to search All Mail when not on a group.
