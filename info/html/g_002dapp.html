11.190.1 g-app Commands

Interactive Command: g-app-mode nil

Major mode for APP interaction

Uses keymap ‘g-app-mode-map

In addition to any hooks its parent mode ‘nxml-mode’ might have run, this mode runs the hook ‘g-app-mode-hook’, as the final step during initialization.’, which is not currently defined.

Interactive Command: g-app-post-entry nil

Post buffer contents as updated entry.

Interactive Command: g-app-publish nil

Publish current entry.

Interactive Command: g-app-put-entry nil

PUT buffer contents as new entry.

Interactive Command: g-app-view (auth-handle feed-url)

View feed using auth credentials in auth-handle.
