11.201 sox

This module defines a convenient speech-enabled interface for editing mp3 and wav files using SoX.

Launching M-x sox creates a special interaction buffer that provides single keystroke commands for editing and applying effects to a selected sound file. For adding mp3 support to sox,

sudo apt-get libsox-fmt-mp3 install

This module can be used independent of Emacspeak.

11.201.1 sox Commands

Interactive Command: sox-add-effect (name)

Adds effect at the end of the effect list

Interactive Command: sox-delete-effect-at-point (&rest rest)

:around advice: ‘ad-Advice-sox-delete-effect-at-point’

Delete effect at point.

(fn)

Interactive Command: sox-edit-effect-at-point nil

Edit effect at point.

Interactive Command: sox-mode nil

An audio workbench for the Emacspeak desktop.

In addition to any hooks its parent mode ‘special-mode’ might have run, this mode runs the hook ‘sox-mode-hook’, as the final step during initialization.

key binding — ——-

Interactive Command: sox-open-file (&rest rest)

:around advice: ‘ad-Advice-sox-open-file’

Open specified snd-file on the Audio Workbench.

(fn SND-FILE)

Interactive Command: sox-play nil

Play sound from current context.

Interactive Command: sox-refresh (&rest rest)

:around advice: ‘ad-Advice-sox-refresh’

Redraw Audio Workbench.

(fn)

Interactive Command: sox-save (save-file)

Save context to file after prompting.

Interactive Command: sox-set-effect (name)

Set effect.

Interactive Command: sox-show-timestamp nil

Show timestamp in stream.

Interactive Command: sox-stop nil

Stop currently playing sound from current context.

11.201.2 sox Options

Variable: User Option sox-edit

Location of SoX utility.

Variable: User Option sox-play
