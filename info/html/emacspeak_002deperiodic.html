11.59 emacspeak-eperiodic

eperiodic produces an interactive periodic table of elements and can be found at http://vegemite.chem.nottingham.ac.uk/~matt/emacs/eperiodic.el

11.59.1 emacspeak-eperiodic Commands

Interactive Command: emacspeak-eperiodic-goto-property-section nil

Mark position and jump to properties section.

Interactive Command: emacspeak-eperiodic-next-line nil

Move to next row and speak element.

Interactive Command: emacspeak-eperiodic-play-description nil

Play audio description from WebElements.

Interactive Command: emacspeak-eperiodic-previous-line nil

Move to next row and speak element.

Interactive Command: emacspeak-eperiodic-speak-current-element nil

Speak element at point.

11.59.2 emacspeak-eperiodic Options
