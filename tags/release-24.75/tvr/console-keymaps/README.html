Customizing The Linux Console Keymap For Emacspeak

The keymap files in this directory configure the Linux console with additional keys that make it convenient to use Emacspeak.

HowTo

  • Copy file emacspeak.kmap.gz to directory /lib/kbd/keymaps/i386/qwerty.
  • Copy file emacspeak.enc.gz to directory /lib/kbd/keytables/i386/include/.
  • As super-user, execute loadkeys /lib/kbd/keytables/i386/qwerty/emacspeak.kmap.gz.
  • To take full advantage of the new keys, load file console.el in your Emacs session.

What You Get

Loading this custom keymap makes the PC keyboard feel like a Unix keyboard --- in addition to making Emacs convenience keys available.

  • Swap Escape and backquote keys.
  • Turn caps-lock into a control key.
  • Make the left-alt and right-alt keys function as Emacs' meta key.
  • Turn the windows-key operate as a hyper modifier key.
  • Turn the second windows key function as a super modifier key.
  • In addition, put hyper on control-;, super on control-' and control-., and ALT on control-,. These latter mappings are useful on laptops.
  • Defines shifted arrow keys.
  • Defines control arrows.
  • Defines shifted tab.
