12.234 espeak-voices

This module defines the various voices used in voice-lock mode by the ESpeak TTS engine.

12.234.1 Espeak-Voices Commands

12.234.1.1 espeak

Command: espeak
C-e d C-e
<fn> d C-e
Start ESpeak engine.
