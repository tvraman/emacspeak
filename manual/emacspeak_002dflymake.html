12.78 emacspeak-flymake

Speech-enable flymake

12.78.1 emacspeak-flymake Options

Variable: User Option voice-flymake-error

Personality used for flymake-error This personality uses voice-highlight whose effect can be changed globally by customizing voice-highlight-settings.

Variable: User Option voice-flymake-note

Personality used for flymake-note This personality uses voice-smoothen whose effect can be changed globally by customizing voice-smoothen-settings.

Variable: User Option voice-flymake-warning

Personality used for flymake-warning This personality uses voice-animate whose effect can be changed globally by customizing voice-animate-settings.
