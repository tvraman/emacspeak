12.124 emacspeak-mines

MINES == Minesweeper game in emacs. The game itself provides a fully keyboard driven interface. In addition, Emacspeak provides

  • Emacspeak-Mines Commands

12.124.1 Emacspeak-Mines Commands

  • emacspeak-mines-beginning-of-row
  • emacspeak-mines-end-of-row
  • emacspeak-mines-goto
  • emacspeak-mines-jump-to-uncovered-cell
  • emacspeak-mines-speak-board
  • emacspeak-mines-speak-cell
  • emacspeak-mines-speak-mark-count
  • emacspeak-mines-speak-neighbors
  • emacspeak-mines-speak-uncovered-count

12.124.1.1 emacspeak-mines-beginning-of-row

Command: emacspeak-mines-beginning-of-row ¶
Move to beginning of row

12.124.1.2 emacspeak-mines-end-of-row

Command: emacspeak-mines-end-of-row ¶
Move to end of row

12.124.1.3 emacspeak-mines-goto

Command: emacspeak-mines-goto (index) ¶
Move to specified cell.

(fn INDEX)

12.124.1.4 emacspeak-mines-jump-to-uncovered-cell

Command: emacspeak-mines-jump-to-uncovered-cell (from-beginning) ¶
Jump to next uncovered cell. With interactive prefix-arg, jump
to beginning of board before searching.

(fn FROM-BEGINNING)

12.124.1.5 emacspeak-mines-speak-board

Command: emacspeak-mines-speak-board ¶
Speak the board.

12.124.1.6 emacspeak-mines-speak-cell

Command: emacspeak-mines-speak-cell ¶
Speak current cell.

12.124.1.7 emacspeak-mines-speak-mark-count

Command: emacspeak-mines-speak-mark-count ¶
Count and speak number of marks.

12.124.1.8 emacspeak-mines-speak-neighbors

Command: emacspeak-mines-speak-neighbors ¶
Speak neighboring cells in sorted order.

12.124.1.9 emacspeak-mines-speak-uncovered-count

Command: emacspeak-mines-speak-uncovered-count ¶
Speak number of uncovered cells.
