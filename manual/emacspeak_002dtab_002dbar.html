12.183 emacspeak-tab-bar

tab-bar == tabs for window configuration. Speech-enable tab-bar interaction. If you have browse-url-new-window-flag set to T to have EWW open Web pages in a new buffer, then set eww-browse-url-new-window-is-tab to nil to avoid leaking tabs.

12.183.1 emacspeak-tab-bar Options

Variable: User Option voice-tab-bar

Personality used for tab-bar This personality uses voice-bolden whose effect can be changed globally by customizing voice-bolden-settings.

Variable: User Option voice-tab-bar-tab

Personality used for tab-bar-tab This personality uses voice-highlight whose effect can be changed globally by customizing voice-highlight-settings.

Variable: User Option voice-tab-bar-tab-inactive

Personality used for tab-bar-tab-inactive This personality uses voice-smoothen whose effect can be changed globally by customizing voice-smoothen-settings.

Variable: User Option voice-tab-line

Personality used for tab-line This personality uses voice-lighten whose effect can be changed globally by customizing voice-lighten-settings.
