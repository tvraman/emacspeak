12.171 emacspeak-sage

Speech-enable sage-shell-mode. This is a major mode for interacting with sage, http://www.sagemath.org/ An Open-source Mathematical Software System.

12.171.1 Emacspeak-Sage Commands

12.171.1.1 emacspeak-sage-describe-symbol

Command: emacspeak-sage-describe-symbol (s)
Describe Sage symbol at point.

(fn S)

12.171.1.2 emacspeak-sage-get-output

Command: emacspeak-sage-get-output
Return most recent Sage output

12.171.1.3 emacspeak-sage-get-output-as-latex

Command: emacspeak-sage-get-output-as-latex
Return most recent Sage output as LaTeX markup.

12.171.1.4 emacspeak-sage-speak-output

Command: emacspeak-sage-speak-output
Speak last output from Sage.
