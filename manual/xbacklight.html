11.211 xbacklight

Provide an emacs front-end to xbacklight sudo apt-get install xbacklight Likely to be most easily used in conjunction with hydra: M-x package-install hydra

11.211.1 xbacklight Commands

Command: xbacklight-0

Set brightness to 0

Command: xbacklight-1

Set brightness to 100

Command: xbacklight-decrement

Decrease brightness by by one step.

Command: xbacklight-get

Get current brightness level.

Command: xbacklight-increment

Increase brightness by by one step.

Command: xbacklight-set (brightness)

Set brightness to specified level. ‘brightness’ is a percentage value.

11.211.2 xbacklight Options

Variable: User Option xbacklight-step

Step-size used when incrementing and decrementing brightness.
