12.207 emacspeak-vdiff

VDIFF == vimdiff Installable from melpa, vdiff enables synchronized movement through diff buffers without resorting to an extra control-panel as is the case with ediff. In addition to speech-enabling interactive commands and setting up face->voice mappings, this module provides commands that speak the current hunk. These are bound in vdiff-mode-prefix-map.

  • emacspeak-vdiff-speak-this-hunk bound to SPC.
  • emacspeak-vdiff-speak-other-hunk bound to C-SPC.
  • emacspeak-vdiff-speak-other-line bound to l.

12.207.1 Emacspeak-Vdiff Commands

12.207.1.1 emacspeak-vdiff-speak-other-hunk

Command: emacspeak-vdiff-speak-other-hunk
Speak corresponding hunk from other buffer.

12.207.1.2 emacspeak-vdiff-speak-other-line

Command: emacspeak-vdiff-speak-other-line
Speak corresponding line from other buffer.

12.207.1.3 emacspeak-vdiff-speak-this-hunk

Command: emacspeak-vdiff-speak-this-hunk
Speak VDiff hunk under point.

12.207.2 emacspeak-vdiff Options

Variable: User Option voice-vdiff-refine-added

Personality used for vdiff-refine-added This personality uses voice-lock-overlay-0 whose effect can be changed globally by customizing voice-lock-overlay-0-settings.

Variable: User Option voice-vdiff-refine-changed

Personality used for vdiff-refine-changed This personality uses voice-lighten whose effect can be changed globally by customizing voice-lighten-settings.
