12.210 emacspeak-widget

This module implements the necessary extensions to provide talking widgets.

  • Emacspeak-Widget Commands

12.210.1 Emacspeak-Widget Commands

  • emacspeak-widget-browse-widget-interactively
  • emacspeak-widget-help
  • emacspeak-widget-summarize-parent
  • emacspeak-widget-summarize-widget-under-point
  • emacspeak-widget-update-from-minibuffer

12.210.1.1 emacspeak-widget-browse-widget-interactively

Command: emacspeak-widget-browse-widget-interactively ¶
Allows you to browse a widget

12.210.1.2 emacspeak-widget-help

Command: emacspeak-widget-help ¶
Speak help for widget under point.

12.210.1.3 emacspeak-widget-summarize-parent

Command: emacspeak-widget-summarize-parent ¶
Summarize parent of widget at point.

12.210.1.4 emacspeak-widget-summarize-widget-under-point

Command: emacspeak-widget-summarize-widget-under-point (&optional level) ¶
Summarize a widget if any under point.
Optional interactive prefix specifies how many levels to go up from current
widget before summarizing.

(fn &optional LEVEL)

12.210.1.5 emacspeak-widget-update-from-minibuffer

Command: emacspeak-widget-update-from-minibuffer (pos) ¶
Sets widget at ‘pos’ by invoking its prompter.

(fn POS)
