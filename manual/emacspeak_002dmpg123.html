11.114 emacspeak-mpg123

This module speech-enables MPG123 MPG123 is an MP3 player.

11.114.1 emacspeak-mpg123 Commands

Command: emacspeak-mp3-playlist-play (playlist &optional dont-shuffle)

Play a playlist. Optional interactive prefix arg says not to shuffle the list. Use command M-x emacspeak-mp3-playlist-skip to skip to the next track.

Command: emacspeak-mp3-playlist-skip

Skip currently playing track.

Command: emacspeak-mp3-playlist-stop

Kill currently playing playlist.

Command: emacspeak-mpg123-backward-minute (arg)

Move back by specified number of minutes.

Command: emacspeak-mpg123-forward-minute (arg)

Forward by ARG minutes.

Command: emacspeak-mpg123-speak-current-time

Speak time in current track.

Command: emacspeak-mpg123-speak-filename

Speak filename of the current song.

Command: emacspeak-mpg123-speak-length

Speak duration of the current song.

Command: emacspeak-mpg123-speak-title

Speak title of the current song.
