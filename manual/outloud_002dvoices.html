12.218 outloud-voices

This module defines the various voices used in voice-lock mode. This module is IBM ViaVoice Outloud specific.

12.218.1 outloud-voices Commands

12.218.1.1 outloud

Command: outloud (&optional device)
C-e d C-o
<fn> d C-o
Select Outloud server.

(fn &optional DEVICE)

12.218.1.2 outloud-32

Command: outloud-32
Select 32-Outloud server.

12.218.2 outloud-voices Options

Variable: User Option outloud-default-speech-rate

Default speech rate for outloud.
