11.122 emacspeak-org

Speech-enable org — Org allows you to keep organized notes and todo lists. Homepage: http://www.astro.uva.nl/~dominik/Tools/org/ or http://orgmode.org/

11.122.1 emacspeak-org Commands

Command: emacspeak-org-bookmark (&optional goto)

Bookmark from org.

Command: emacspeak-org-popup-input

Pops up an org input area.

Command: emacspeak-org-table-speak-current-element

echoes current table element
